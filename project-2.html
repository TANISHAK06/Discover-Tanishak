Project Overview

This calculator was meticulously designed to ensure seamless functionality across all devices, from mobile phones to desktops.

It features intuitive arithmetic operations, a clear interface with delete functionality, and a dynamic display that resets elegantly after calculations. Built using HTML, CSS for responsive design, and JavaScript for interactive functionality, this project not only showcases my technical proficiency but also highlights my commitment to user-centric design and accessibility. Explore the project on GitHub to see how it enhances usability through responsive web design and a customizable theme toggle.
